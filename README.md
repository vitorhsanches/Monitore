# Monitore

Monitore é um aplicativo colaborativo para o registro de ocorrências de manutenção urbana. Cidadãos podem relatar problemas como calçadas danificadas e rampas de acesso em mau estado. O objetivo é melhorar a infraestrutura urbana de forma acessível e eficiente, conectando cidadãos e gestão pública.
